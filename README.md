# WaveRunner 

WaveRunner is a 2D game developed using the Godot Engine.

The player controls a character that must survive waves of enemies in a dynamic environment.  
This project was created as a personal learning project during the summer.

---

##  Built With
- Godot Engine
- GDScript

---

##  How to Run the Project
1. Open Godot
2. Click on **Import**
3. Select the file `project.godot`
4. Run the main scene

---

##  Controls
- Arrow keys / WASD: Move the player
- Escape: Quit the game

---

##  Project Structure
- `scene/` : Game scenes
- `art/` : Sprites, sounds, and assets
- `fonts/` : Fonts used in the game
- `*.gd` : Game scripts

---

## Notes
This repository contains the source code and assets of the game.  
Build files (`.exe`, `.pck`) are not required to run the project inside Godot.
